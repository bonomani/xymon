Canonical dependency list for packagers
======================================

Build (Linux)
-------------
Ubuntu-latest (GitHub Actions):
  Base:
    - build-essential
    - libssl-dev
    - libpcre3-dev
    - librrd-dev
    - libtirpc-dev
  Server only:
    - libc-ares-dev
  LDAP (when enabled):
    - libldap-dev
  Compiler:
    - clang

Non-debian profile:
  Base:
    - clang
    - build-essential
    - libssl-dev
    - libpcre3-dev
    - librrd-dev
    - libtirpc-dev
  LDAP (when enabled):
    - libldap2-dev

Build (BSD)
-----------
Common (all BSDs):
  - gmake
  - cmake
  - pcre

FreeBSD (pkg):
  Server only:
    - c-ares
  LDAP (when enabled):
    - openldap-client

NetBSD (pkgin):
  Server only:
    - libcares
  LDAP (when enabled):
    - openldap-client

OpenBSD (pkg_add):
  Server only:
    - libcares
  LDAP (when enabled):
    - openldap-client

Runtime
-------
Common:
  - sendmail (or equivalent)
Recommended:
  - fping
  - perl
  - python
Optional:
  - OpenSSL (SSL checks)
  - OpenLDAP client (LDAP checks)
  - net-snmp (SNMP collector)
  - c-ares

Version notes
-------------
Linux:
  - Ubuntu-latest package names shown; adjust for other distros/versions.
  - Non-debian profile reflects CI; verify on your distro/version.
BSD:
  - Package naming varies by OS release; verify for your FreeBSD/NetBSD/OpenBSD version.
