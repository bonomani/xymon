Legacy Migration Docs Index
===========================

Purpose
-------
This folder contains the authoritative documents for the legacy CMake migration.
Use the guide below to pick the right entry point.

Quick Start
-----------
- If you need the plan/checklist: `10-PLAN.md`
- If you need to run the steps: `20-RUNBOOK.md`
- If you need the legacy reference contract: `30-REFERENCE.md`
- If you want the latest status summary: `40-STATUS.md`

Contents
--------
- README.md: naming convention and usage
- 10-PLAN.md: validation checklist and acceptance criteria
- 20-RUNBOOK.md: step-by-step commands and validation procedure
- 30-REFERENCE.md: canonical legacy install layout and constraints
- 40-STATUS.md: condensed progress / current status
